# Lottie Book

Deno + Vue3で開発をしています。 開発環境はdocker composeを使っています。

## 開発コマンド

環境セットアップ

```sh
make setup
```

コンテナの中に入る

```sh
make run
```

コンテナを立ち上げて、ブラウザで`localhost:5173`を開く

```sh
make up
```

コード整形（コンテナの中で実行してね）

```sh
deno fmt
```
